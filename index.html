pytest mini-workshop

Welcome to the mini-workshop on pytest as part of CW18.

You have two options for how to set run the examples as part of this workshop. You can either run the code locally on your laptop or connect to the web-based terminal.

On your laptop

To run the workshop on your laptop you'll need:

  • Python â‰¥ 3.3
  • Git
  • The ability to run a command-line
  • A text editor
This is likely easy if you are running Linux or MacOS.

Open a terminal and run the following:

git clone https://github.com/milliams/pytest_miniworkshop.git
cd pytest_miniworkshop
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
at which point you are ready to start running pytest. Try it!

Web-based terminal

If you don't have easy access to a command-line or a Python installation on your laptop then try connecting to the web-based terminal service.

Point your browser to pytestworkshop1.westeurope.cloudapp.azure.com. To find out what username and password you need, go to the Google spreadsheet, choose an unclaimed username and put your name next to it. I'll give the passwords out in-person.

Once you are logged in, do the following to make sure you are in a working environment:

cd pytest_miniworkshop
source venv/bin/activate
pytest
You are then ready to go!
